
# Simple Web Applaication For FHIR

## Installation
1. clone git project
```bash
  git clone https://github.com/piyapong-mun/HealthFHIR.git
```
2. got to */HealthFHIR*
```bash
  cd /HealthFHIR
```
3. update node module
```bash
  npm update
```
4. Config FHIR endpoint
    - go to `HealthFHIR/script/fhir-config.js`
    - replace URL with your FHIR endpoint
5. start server
```bash
  npm start
```
